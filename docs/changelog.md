# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-03

### Added
- Initial release of MCP Experto Filesystem.
- Implementation of advanced filesystem tools (`replace`, `grep_search`, etc.).
- Integration with Local Telemetry (Prometheus, Grafana, Loki).
- Strategic Help system with markdown templates.
- Unified technical documentation and standards.

### Changed
- Documentation structure reorganized: moved technical references to `./references/` and focused `./docs/` on user-facing guides.
